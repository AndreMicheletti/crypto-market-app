# Crypto Marker App

Simple native App to track the crypto currency market values.

## Techstack

- [React-Native 0.51](https://facebook.github.io/react-native/)
- [Redux](https://redux.js.org/)
- Native-Base
- axiosjs

## Features

 - [] Fetch Crypto Currency via APIs
 - [] View details
 - [] Search and Filter
